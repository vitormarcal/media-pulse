# Frontend V2

Home editorial do Media Pulse em Nuxt 4, construída sem dependência do frontend legado.

## Setup

Instale as dependências:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Development

Suba o backend em paralelo e inicie o Nuxt em `http://localhost:3000`.

Se a API estiver em outra origem, defina `NUXT_PUBLIC_API_BASE`.

```bash
NUXT_PUBLIC_API_BASE=http://localhost:8080 npm run dev
```

## Build

```bash
npm run dev
```

```bash
npm run build
```
